Gradwell PEAR channel

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover pear.gradwell.com
Listing available packages: 
pear remote-list -c Gradwell
Installing a package: 
pear install Gradwell/package_name
Installing a specific version/stability: 
pear install Gradwell/package_name-1.0.0
pear install Gradwell/package_name-beta
Receiving updates via a feed: 
http://pear.gradwell.com/feed.xml
